Detailed Description

Holder for Views used in the LiteListDemoActivity.MapAdapter. Once the the map field is set, otherwise it is null. When the onMapReady(com.google.android.gms.maps.GoogleMap) callback is received and the com.google.android.gms.maps.GoogleMap is ready, it stored in the map field. The map is then initialised with the NamedLocation that is stored as the tag of the MapView. This ensures that the map is initialised with the latest data that it should display.
